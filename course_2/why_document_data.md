# Why Document Data?
Datasets shape
- Who is represented
- Who is excluded
- How cultural knowledge is treated

Documentation provides 
- Where data came from
- Whether consent was given
- Data access/restrictions
- Ownership
- Benefit sharing programs

**Data Card**: answers questions about 
- governance
- privacy
- consent
- ownership
- resprersentation

    to build
- trust
- accountability
- value alignment

### Exercise
You've found an imaginary dataset w/o any documentation. What questions would you ask?
1. Who owns it? Where is the licensing agreement?
2. Why did they create this dataset? What was their goal?
3. How was the data prepared?
4. What potential risks could this dataset introduce? Is there a chance of malicious data being included?
5. Does it contain sensitive data? PII? Credit card numbers? Passwords? If not, how was it scrubbed?
